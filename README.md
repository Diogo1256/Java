# 🧮 Calculadora de Bhaskara

Projeto desenvolvido em Java para resolver equações do segundo grau utilizando a fórmula de Bhaskara.

## 🚀 Funcionalidades

- Entrada dos coeficientes A, B e C
- Cálculo do Delta (Δ)
- Cálculo das raízes da equação
- Exibição dos resultados no terminal

## 📚 Conceitos utilizados

- Variáveis
- Operadores matemáticos
- Estruturas condicionais
- Entrada de dados com Scanner
- Classe Math (`Math.sqrt()`)

## 💻 Tecnologias

- Java
- IntelliJ IDEA
